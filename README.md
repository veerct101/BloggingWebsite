# BloggingWebsite
Built a full-stack blogging platform using Node.js, Express.js, MongoDB, and EJS. Features include JWT-based authentication, secure password hashing, blog creation, image uploads with Multer, commenting functionality, CRUD operations, and dynamic server-side rendering following MVC architecture.


## About The Project
This is a full-stack Blogging Platform built using Node.js, Express.js, MongoDB, and EJS. The application allows users to register, log in securely using JWT authentication, create and publish blog posts, upload cover images using Multer, and interact through a commenting system. Passwords are protected using hashing and salting techniques, while MongoDB and Mongoose handle data storage and management. The project follows the MVC architecture and demonstrates key backend concepts such as authentication, CRUD operations, file handling, middleware, database integration, and server-side rendering.


## Features
- User Registration & Login
- JWT-based Authentication
- Password Hashing & Salting
- Create, Read, and Manage Blogs
- Cover Image Uploads using Multer
- Comment System
- MongoDB Database Integration
- Server-Side Rendering with EJS
- MVC Architecture
- Responsive UI with Bootstrap


## Tech Stack

**Frontend:** HTML, CSS, Bootstrap, EJS
**Backend:** Node.js, Express.js
**Database:** MongoDB, Mongoose
**Authentication:** JWT, SHA-256 Hashing


## User Guide

### 1. Register an Account
- Navigate to the Sign Up page.
- Enter your name, email, and password.
- Create your account.

### 2. Log In
- Use your registered email and password to log in.
- After successful authentication, you will be redirected to the home page.

### 3. Create a Blog Post
- Click on **Add Blog**.
- Enter the blog title and content.
- Upload a cover image.
- Submit the form to publish your blog.

### 4. Browse Blogs
- View all published blogs on the homepage.
- Click on a blog to read its full content.

### 5. Comment on Blogs
- Open any blog post.
- Add your comment in the comment section.
- Submit to interact with other users.

### 6. Manage Content
- Logged-in users can create and publish multiple blog posts.
- Uploaded images are displayed as blog cover images.

**File Uploads:** Multer

**Version Control:** Git & GitHub
